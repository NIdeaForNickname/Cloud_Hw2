# Cloud_Hw2

1) Используя командную строку AWS CLI, покажите информацию о пользователях и группах.
В качестве результата домашнего задания: скриншоты из AWS CLI, где можно увидеть полученную информацию.

2) Используя CloudShell. Создайте следующую иерархию
folder1
-subfolder
-- textFile.txt > "Hello from subfolder"
folder2
-subfolder
--subsubfolder

Вывести иерархию с помощью tree

3) Запустить инстанс Линукса, подключиться с помощью Putty. Скриншот
4) Запустить инстанс Виндоуса, подключиться с помощью средства "Удаленный рабочий стол". Скриншот
